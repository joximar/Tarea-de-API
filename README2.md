===RESTAURAR TODOS LOS PAQUETES===

npm install

======INSTALAR PRISMA DENTRO DE TU PROYECTO YA REALIZADO==

npm install prisma --save-dev
npm install @prisma/client

======INSTALANDO PRISMA CLI ======

npx prisma

=======CREAR LA CONFIGURACION INICIAL DE PRISMA=========

npx prisma init

==========CREAR LA PRIMERA MIGRACION =========

npx prisma migrate dev --name init      LA PRIMERA MIGRACION ES NOMBRE INIT

===========GENERAR EL CLIENTE DE PRISMA============

npx prisma generate

==========ejecutar o lanzar el proyecto===========
npm run start:dev
